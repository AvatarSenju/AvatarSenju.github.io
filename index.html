About Me

I'm a software engineer who has contributed to projects ranging from the Rust programming language (merged PR to rust-lang/rust) to building object tracking and face detection systems at LNMIIT's Phoenix lab. I've shipped production code across Python, C++, Rust, and JavaScript.

My open source journey started in 2017 — with early Hacktoberfest contributions, then progressing to substantial work: adding invoice parsing templates to invoice2data, contributing to FOSSASIA and EbookFoundation, building an autonomous RoboSoccer system, and ultimately landing a PR in the Rust compiler bootstrap toolchain.

I'm an Arctic Code Vault Contributor, Pull Shark x3, and Pair Extraordinaire x3. I reached the SIH 2020 Finals as part of Team SAAHASi.
